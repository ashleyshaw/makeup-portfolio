
# Contributing Guide

Thank you for your interest in contributing to the Ash Shaw Makeup Portfolio project!

## How to Contribute

- This repository uses the default issue and pull request templates from the [lightspeedwp/.github](https://github.com/lightspeedwp/.github) community health repository. Do not add or edit local templates unless you want to override the defaults.
- **Code Style:**
  - Follow the coding standards and configuration in `.editorconfig`, `.prettier.config.cjs`, `.eslint.config.cjs`, and related files.
- **Documentation:**
  - Update or add documentation in the appropriate `.md` files in `.github/instructions/`.

## Project-Specific Guidelines

- All contributions should be tailored to the needs and standards of the Ash Shaw Makeup Portfolio project.
- If you are unsure about a change, open an issue or discussion first.

## Code of Conduct

By participating, you agree to abide by our [Code of Conduct](./CODE_OF_CONDUCT.md).
